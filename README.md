# Excel Importer

A Flutter application for importing Excel (.xlsx) and CSV files, converting them to JSON format with a GitHub-inspired UI.

## Features

- **File Import**: Support for `.xlsx` and `.csv` files
- **JSON Conversion**: Convert spreadsheet data to structured JSON format
- **CSV Auto-detection**: Automatically detects Tab or comma delimiters in CSV files
- **WPS Compatibility**: Parses xlsx files generated by WPS Office
- **Import History**: View and manage previously imported files
- **Export**: Export data back to Excel format
- **Internationalization**: Chinese/English language support

## Getting Started

### Prerequisites

- Flutter SDK 3.11.1 or higher
- Dart SDK 3.11.1 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/szhua/testvsFullter.git

# Navigate to project directory
cd excel_importer

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Build

```bash
# Android
flutter build apk

# iOS
flutter build ios

# Web
flutter build web
```

## Tech Stack

| Package | Purpose |
|---------|---------|
| `syncfusion_flutter_xlsio` | Excel file creation/export |
| `archive` + `xml` | xlsx file parsing (reading) |
| `csv` | CSV parsing |
| `file_picker` | File selection |
| `shared_preferences` | Local data persistence |
| `provider` | State management |
| `intl` | Date formatting and i18n |
